# UpRight

A Chrome Extension that organizes your browser tabs in a vertical sidebar for improved tab management and navigation.

## Overview

UpRight provides a left-side vertical tab bar in Chrome, making it easier to:
- View and manage many open tabs at once
- See full tab titles without truncation
- Navigate between tabs quickly
- Organize your browsing workflow

## Features (Planned)

- Left sidebar displaying all open tabs vertically
- Tab switching by clicking on tabs in the sidebar
- Visual indication of the active tab
- Close tabs from the sidebar
- Tab favicon display
